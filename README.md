# ban3110-homework-7

## Overview

This project demonstrates how Python can be used to connect multiple APIs together to analyze keyword trends and marketplace product data.

The notebook retrieves trending search keywords from Google using a RapidAPI service. It then uses those keywords to search Amazon product listings and identify the top-rated products associated with those search terms.

The final result is exported to a Google Sheets document so the data can be easily viewed and analyzed.

## What the Program Does

1. Calls a Google keyword API to retrieve trending search keywords.
2. Selects the top three keywords returned by the API.
3. Searches Amazon products using each keyword.
4. Ranks the returned products based on:

   * Highest star rating
   * Highest number of ratings (as a tiebreaker)
5. Selects the top three products for each keyword.
6. Extracts product details such as title, price, rating, and product URL.
7. Organizes the results into a pandas DataFrame.
8. Exports the final dataset to Google Sheets.

The program generates a dataset containing:

* Google search keyword
* Amazon product title
* Product price

This output helps connect search demand from Google with real product offerings on Amazon.
